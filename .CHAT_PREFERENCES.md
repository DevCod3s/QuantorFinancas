# Preferências de Chat do Projeto

Este arquivo registra a preferência do mantenedor/do usuário para interações com assistentes e agentes no contexto deste repositório.

- Preferência de idioma: Português (Brasil)
- Instrução para novos chats/assistentes: "Responda em Português (Brasil)."

Observações:
- Este arquivo serve apenas como documentação local do projeto; ele não pode forçar outros agentes externos a seguirem a regra.
- Recomenda-se reproduzir a instrução no início de qualquer novo chat com assistentes para garantir respostas em Português (Brasil).
